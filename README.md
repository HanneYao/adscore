# 广告类型与工种等级评分系统 📊

一个用于生成广告效果评分的静态网页工具，支持多种广告类型和工种等级配置，基于概率分布生成随机评分。

## 🌟 项目介绍
- 支持 9 种广告类型（视频类、图文类、裂变类等）
- 可配置工种等级（S/A/B/C 级），自动计算总费用
- 可视化评分分布预测（均值、标准差、偏度）
- 生成符合概率分布的随机评分，支持历史记录查看

## 🌐 在线访问
[广告类型与工种等级评分系统](https://hanneyao.github.io/adscore)  

## 🚀 功能特点
1. **广告类型选择**：9 种主流广告类型，按需切换
2. **工种配置**：区分必选/可选工种，支持等级调整
3. **数据可视化**：Chart.js 绘制评分分布图表，直观展示概率
4. **费用计算**：实时计算所选工种等级的总费用
5. **随机评分**：基于偏态正态分布生成真实可信的评分
6. **响应式设计**：适配电脑、平板、手机等多种设备

## 📋 使用说明
1. 打开在线链接，选择广告类型（比如「视频类」）
2. 配置各工种等级（必选工种不可跳过，可选工种可选择「无」）
3. 查看评分分布预测图表和费用汇总
4. 点击「生成随机评分」按钮，获取符合当前配置的随机评分
5. 查看历史评分记录（最多保留 5 条）
